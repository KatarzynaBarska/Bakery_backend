# WIRTUALNA PIEKARNIA

Zobacz projekt [Virtual Bakery](https://devmentor.pl).

Aplikacja webowa, mająca na celu zademonstrowanie moich umiejętności pisania kodu.
Umożliwia zamawianie świeżego pieczywa oraz daty dostawy.

**Właściwości**:
- wybór pieczywa
- wybór ilości
- wybór daty
